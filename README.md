# OIBSIP - Oasis Infobyte Python Internship

This repository contains the Python internship tasks completed as part of the Oasis Infobyte Internship Program.

## Tasks Completed

### Task 1: BMI Calculator
- Calculates Body Mass Index (BMI)
- Displays BMI category

### Task 2: Password Generator
- Generates strong random passwords
- User can choose password length

### Task 3: Weather App
- Displays current weather information
- Fetches weather data using an API
