Weather App

Description

This Weather App fetches real-time weather information for any city using the OpenWeatherMap API. Users can enter a city name and get details such as temperature, humidity, and weather conditions.

Features

- Get real-time weather data
- Search weather by city name
- Displays temperature in Celsius
- Shows humidity percentage
- Shows weather condition description

Technologies Used

- Python
- Requests Library
- OpenWeatherMap API

How to Run

python weather_app.py

Installation

Install the required library:

pip install requests

Example

Input:

Enter city name: Brahmapur

Output:

Weather Report
City: Brahmapur
Temperature: 31.5 °C
Humidity: 70 %
Condition: scattered clouds
