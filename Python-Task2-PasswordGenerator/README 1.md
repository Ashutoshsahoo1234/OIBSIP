Password Generator

Description

This project generates a strong random password based on the length entered by the user. It uses Python's built-in modules to create secure passwords containing letters, numbers, and special characters.

Features

- Generates random passwords
- User can choose password length
- Includes uppercase and lowercase letters
- Includes numbers and special characters
- Simple and easy to use

Technologies Used

- Python
- random module
- string module

How to Run

python password_generator.py

Example

Input:

Enter password length: 10

Output:

Generated Password:
A#7kP!2mX@
